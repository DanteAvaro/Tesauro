# Tesauro - Herramienta de Gestión de Términos

Este programa "Tesauro" es una herramienta de gestión de términos que permite a los usuarios agregar, buscar, eliminar y exportar términos y definiciones. La aplicación está diseñada para ofrecer una interfaz gráfica amigable y funcional.

## Requisitos del sistema
- **Sistema Operativo**: Windows 10 o superior.
- **Python**: No se requiere instalar Python para los usuarios finales, ya que el programa viene empaquetado como un archivo ejecutable (.exe).
- **Antivirus**: Algunos antivirus pueden detectar falsos positivos al descargar o ejecutar el archivo .exe. Asegúrate de permitir su ejecución o añadirlo a las excepciones si es necesario.

## Instalación

### Paso 1: Descargar el archivo .exe
Dirígete a la sección "Releases" en el repositorio de GitHub y descarga el archivo `Tesauro.exe` más reciente.

### Paso 2: Ejecutar el programa
1. **Ejecutar el archivo .exe**: Una vez descargado, haz doble clic sobre el archivo `Tesauro.exe` para iniciar la aplicación. No es necesario instalar nada adicional.
2. Si ves una advertencia de seguridad de Windows SmartScreen o de tu antivirus, selecciona "Más información" y luego haz clic en "Ejecutar de todos modos".

## Cómo usar el programa en tu PC

### Recomendaciones:
- **Accesos directos**: Para un acceso rápido, puedes crear un acceso directo del archivo `Tesauro.exe` en tu escritorio o en el menú de inicio.
- **Permisos de ejecución**: Asegúrate de que el archivo `Tesauro.exe` tenga permisos de escritura en la carpeta en la que lo guardes. Esto es necesario para que se puedan guardar las entradas que crees. Si notas que los cambios no se están guardando, intenta ejecutar el programa como administrador (clic derecho sobre el archivo .exe > Ejecutar como administrador).
- **Antivirus y Seguridad**: Si tu antivirus bloquea el programa o te impide ejecutarlo, revisa las configuraciones del antivirus y añade el archivo `Tesauro.exe` a las excepciones de programas permitidos.
- **Guardar en carpetas accesibles**: Guarda el archivo en una carpeta donde tengas permisos de lectura y escritura, como en tu carpeta de Documentos, para evitar problemas al guardar datos.

## Características del programa

### 1. **Agregar términos**
   - Rellena los campos "Palabra", "Descripción" y "Etiquetas" en la interfaz.
   - Elige el tipo de entrada (definiciones o ideas).
   - Haz clic en el botón "Agregar Entrada" para guardar el término.

### 2. **Buscar términos**
   - Introduce una palabra en el campo "Palabra" y haz clic en "Buscar Entrada" para encontrar y ver las entradas existentes.

### 3. **Eliminar términos**
   - Introduce una palabra en el campo "Palabra" y haz clic en "Eliminar Entrada" para eliminar la entrada correspondiente.

### 4. **Exportar datos**
   - Haz clic en el botón "Exportar Datos" y elige si deseas exportar tus términos a un archivo **CSV** o **HTML**.

## Exportación de Datos

### Exportar a CSV:
Este formato es ideal para trabajar con hojas de cálculo como Excel. Los datos se guardan en un archivo `.csv` que contiene la palabra, descripción, etiquetas y tipo.

### Exportar a HTML:
El formato HTML permite visualizar las entradas en un formato más visual y organizado. El archivo `.html` puede abrirse directamente en cualquier navegador web.

## Problemas comunes

### 1. **El antivirus bloquea el archivo .exe**
   - Algunos antivirus pueden detectar el archivo .exe como sospechoso debido a la forma en que fue empaquetado. Si confías en el archivo, añádelo a las excepciones de tu antivirus.

### 2. **No se guardan los cambios**
   - Asegúrate de tener permisos para escribir en la carpeta donde guardas los datos. Si es necesario, ejecuta el programa como administrador.
